KEY METRICS REFERENCE
====================================================================================

DISTANCE CALCULATION (Rectilinear/Manhattan Distance)
• Method: |x2-x1| + |y2-y1| (horizontal + vertical distance)
• Rationale: Manufacturing facilities use grid-based layouts
• Per Trip: 244 ft (B1→A→B2→C→D→I→J)
• Average Transfer: 40.7 ft
• Weekly (Year 2): 1,206 miles @ 26,100 trips
• Weekly (Year 3): 1,262 miles @ 27,300 trips
• Weekly (Year 4): 1,317 miles @ 28,500 trips

FACILITY DIMENSIONS
• All years: 298 ft width × 108 ft height = 32,184 ft²
• Process flow: B1 → A → B2 → C → D → I → J
• Gap between areas: 5 ft
• No facility expansion needed Year 2-4

MACHINE COST ESTIMATES
• B1/B2: $150,000 each
• A: $200,000
• C: $180,000
• D: $175,000
• I: $220,000
• J: $195,000
• Average cost per machine: $175,833-$176,316
